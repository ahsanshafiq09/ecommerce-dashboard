# 📊 E-commerce Analytics Dashboard

## 🚀 Overview
A full-stack data analytics project that processes e-commerce data and displays insights through a web dashboard.

## 🛠 Tech Stack
- Python (pandas, Flask)
- JavaScript (Chart.js)
- HTML/CSS

## 📊 Features
- Total revenue analysis
- Top category & customer insights
- Monthly performance tracking
- Interactive charts

## ▶️ How to Run

### Backend
cd backend
pip install -r requirements.txt
python api.py

### Frontend
Open frontend/index.html in browser

## 💡 Key Skills Demonstrated
- Data analysis & ETL
- API development
- Data visualization
- Full-stack integration
